ERP Odoo - Gestion de Stock


== Fonctionnalités
-  Ajouter un produit
-  Vendre un produit
- Afficher la liste des produits

== Structure du projet
code/
├── ajouterproduit.php
└── style.css


== Technologies utilisées
- PHP
- Odoo API (JSON-RPC)
- HTML / CSS
- cURL

== Installation
1. Cloner le dépôt
2. Placer dans htdocs (XAMPP)
3. Configurer les identifiants Odoo
4. Lancer : http://localhost/.../code/ajouterproduit.php
